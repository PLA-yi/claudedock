## 新增文件

### internal/controlplane/http/admin_host_files.go
- `AdminHostFilesHandler` 结构体 + `NewAdminHostFilesHandler(logger)` 构造函数
- `List()` 方法实现：
  - 从 query 参数读取 `path`
  - 校验非空、以 `/` 开头、不含 `..`
  - 敏感目录黑名单：`/proc`, `/sys`, `/dev`, `/boot`, `/etc/ssh`, `/root/.ssh`
  - `os.Stat` 确认存在且为目录
  - `os.ReadDir` 读取，仅返回子目录名（过滤文件、symlink）
  - 返回 JSON `{ "entries": [...] }`

### internal/controlplane/http/admin_host_files_test.go
7 条测试全部 PASS：
- `TestAdminHostFiles_NormalRequest` — 正常返回子目录列表
- `TestAdminHostFiles_PathTraversal` — `../etc/passwd` → 400
- `TestAdminHostFiles_SensitiveDir` — `/proc` 等 → 403
- `TestAdminHostFiles_NotFound` — 不存在路径 → 404
- `TestAdminHostFiles_NotADirectory` — 普通文件 → 400
- `TestAdminHostFiles_EmptyPath` — 空 path → 400
- `TestAdminHostFiles_RelativePath` — `foo/bar` 等 → 400

### web/admin/src/hooks/use-host-files.ts
- `useHostFiles(path)` hook，使用 `useQuery`
- `queryKey: ["host-files", path]`
- `enabled: path.length > 0 && path.startsWith("/")`
- `staleTime: 30_000`

### web/admin/src/components/hosts/path-autocomplete.tsx
- Props: `value`, `onChange`, `placeholder`, `disabled`, `className`
- 行为：
  - 输入框获得焦点且 value 以 `/` 开头时展开下拉
  - 失去焦点延迟 150ms 关闭（允许点击选项）
  - 键盘支持：↓/↑ 移动高亮，Enter 选中，Esc 关闭
  - 加载状态显示 `Loader2` 旋转图标
  - 空结果显示 "无可用子目录"
  - 选中后自动补 `/` 并追加子目录名
