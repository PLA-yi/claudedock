# Phase 49 Plan 04: LEAK-04 IPv6 阻断

## Goal

证实 worker 容器全 IPv6 出网均被阻断：

- `curl -6 --max-time 3 https://ipv6.google.com` 必须失败。
- 容器内 `cat /proc/sys/net/ipv6/conf/all/disable_ipv6` 必须返回 `1`（双保险，已在 Phase 47 Plan 02 I6 锁定）。
- 同时 `default/disable_ipv6` 也必须为 `1`。
- host eth0 IPv6 抓包：若 worker 有 IPv6 地址，BPF `ip6 and src <workerIPv6>` 必须 0 包；无 IPv6 地址 → 跳过抓包断言（IPv6 stack 整个被关）。

`TestLeak_04_IPv6_BlockedByHostEth0`。

  - `*GoldenPath.ReadProcFile(ctx, path string) (string, error)` —— 通用 `cat /proc/...`。
- `tests/e2e/leak/leak_04_ipv6_test.go`（带 tag，新文件）。

## Steps

1. **`*GoldenPath.CurlIPv6`**：
   - `docker exec <worker> curl -6 -sS --max-time 3 <url>`。
   - 解析：
     - exit 0 → `Blocked=false`（IPv6 出网成功，leak）。
     - exit 6 (Couldn't resolve host) / 7 (Couldn't connect) / 28 (timeout) → `Blocked=true, Reason="curl_v6_failed"`。
     - stderr 含 `Network unreachable` / `disabled` → `Blocked=true`。

2. **`*GoldenPath.ReadProcFile`**：
   - `docker exec <worker> cat <path>`。
   - 返回 trim 后的 stdout；exit !=0 → err。

3. **主用例 `TestLeak_04_IPv6_BlockedByHostEth0`**：
   - 双保险断言：`g.ReadProcFile(ctx, "/proc/sys/net/ipv6/conf/all/disable_ipv6")` == `"1"`；同样 `/default/disable_ipv6`。
   - `g.CurlIPv6(ctx, "https://ipv6.google.com")` → `curlResult.Blocked == true`。
   - host eth0 抓包：先尝试拿 worker IPv6 地址（`docker inspect -f '{{.NetworkSettings.GlobalIPv6Address}}'`）；为空 → 跳过抓包断言（worker 完全没 IPv6 stack，符合预期）；非空 → BPF `ip6 and src <ipv6>`，count=5 必须 0 包。
   - 任一 fail → t.Fatalf。

4. **darwin 兜底**：build tag。

## Acceptance

darwin：纯函数单测（`ClassifyLeakProbe` 复用 + ReadProcFile path 字面量校验）PASS。Linux：用例 PASS。

## Risk & Mitigation

| 风险 | 缓解 |
|------|------|
| worker 没装 curl IPv6 支持 | curl 4.x 默认带；ubuntu 24.04 默认包含。grep 已确认镜像装 curl。 |
| `disable_ipv6=1` 导致 curl 在解析阶段就失败而 exit 6（无 IPv6 地址） | 这正是预期；归为 Blocked=true。 |
| host eth0 IPv6 抓包 oracle 在 IPv4-only runner 上抓不到 | 用例先看 worker 有无 IPv6；无 → 跳过抓包，仅靠双 disable_ipv6 + curl 失败两层断言。 |
| 当前 worker.go 已显式 sysctl `disable_ipv6=1` (worker.go:229-230)，预期 PASS | 预期 LEAK-04 在 source-of-truth 层已就位，本 plan 用例应直接 Pass，不进 Phase 51 GAP。 |
